Basic Algorithm- Sorting

Posted on 2021-01-27 Edited on 2021-02-10 In Sorting , Basic-Algorithm

前序，中序，后续

Pattern of different order

1. Quick Sort

  • 确定分界点，可以随便找到一个值
    • x = q[left], q[(l+r)/2], q[right], random
  • 调整区间，所有<=x 在左半边， 所有>=x 在右半边， 其中每个元素的位置都可能发生改变， 相等可以不用调整
  • 递归，处理左右两端，排序左边，排序右边
    • 前序递归，从左右两大块开始
    • 先分成两个大块，所有<=x 在左半边， 所有>=x 在右半边
    • 再进一步各自分成两块
    • 直到两边各只有一个元素

另一个选择

举例：

  • int[] q = [1,2] i=0, j = 1, privot = 1
  • 移动指针， i = 0, j = 0
  • quickSort(q, l,-1);
  • quickSort(q, 0, 1); 不断循环quickSort(q, 0, 1)

2. Merge Sort

  • 确定分界点(index)： mid = (l+r)/2
  • 递归排序 left and right
    • mergeSort([] q, left, mid)
    • mergeSort([] q, mid + 1, right)
    • 后序递归
    • 从单个元素开始
    • 一直递归到两边各只有一个元素，比较两个元素的大小，合并，存在tmp中， 写入 int[] q
    • 另一边上的两个元素，比较，合并
    • 再比较，合并（4个元素）
    • 再进一步比较，合并
  • 归并： 合二为一
    • 双指针算法
    • 需要一个额外空间， index: k = 0
    • i = left 左半边的起点
    • j = mid + 1 右半边的起点
