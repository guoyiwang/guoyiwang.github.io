高精度加法

  • 从低位开始，每个存在数组里面

高精度减法

  • A >= B, 直接算
  • A < B, -(B-A)
  • 最高位不用再借位

高精度乘法

  • A是比较大的数，B作为比较小的数，是作为一个整体与A相乘
  • A: 123, B:25
  • t1 = 3*25 + 0(初始化的借位0), t1%10 = 5, result.add(5), t1 /= 10, t1 = 7
  • t2 = 2*25 + t1 = 57, result.add(7), t2 /=10, t2 = 5;
  • t3 = 1*25 + 5 = 30, result.add(0), t3 /=10, t3 = 3;
  • 最后一位，if(t3!=0), result.add(3)
  • 即3075

高精度除法

  • 高精度整数 / 低精度整数
  • 123 / 5
  • t = 0, t = (t*10+1) = 1, result.add(1/5), t = t%5 = 1
  • t = 1, t = (t*10+2) = 12, result.add(12/5), t = 12%5 = 2
  • t = 2, t = (t*10+3) = 23, result.add(23/5), t = 23%5 = 3
  • result = 24, 余数为3
