PriorityQueue

  • Based on a priority heap
  • Internal data structure is not ordered, is a heap.
  • Method
    • poll: Retrieves and removes the head of this queue, or returns null if this queue is empty.
    • add: Inserts the specified element into this priority queue.
      • comes from Collection
      • always returns true and throws an exception if it can’t add the element
    • offer: Inserts the specified element into this priority queue.
      • comes from Queue
      • is allowed to return false if it can’t add the element.

Deque (Double Ended Queue)

  • You can enqueue and dequeue from both ends of a Java Deque
  • You can use a Deque as both a queue and a stack
  • Add Element to Deque：add elements to both the beginning and end of a Deque
    • add()
    • addLast()
    • addFirst()
    • offer()
    • offerFirst()
    • offerLast()
    • push(): adds an element to the beginning (head) of a Java Deque method
  • Peek at Element in Deque
    • peek()
    • peekFirst()
    • peekLast()
    • getFirst()
    • getLast()
